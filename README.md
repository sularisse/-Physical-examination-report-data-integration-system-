隊伍名稱：咩咩隊
- 作品名稱：-Physical-examination-report-data-integration-system-
- 主題領域：醫學檢驗/醫療資訊
- 使用者角色：病人/醫師/醫事人員:醫檢師/放射師/理學人員
- 核心 FHIR Resources：Patient、Observation
--------------------------------------------------------------------
-主題及應用情境:將各醫院的體檢報告整合，讓不同醫院的醫事人員以及病人得以檢閱及編輯
-程式碼及執行結果:位於index.html中
-說明文件:位於S1大健康_咩咩隊_規劃文件.pdf
-程式可能延伸應用:可進一步整合病患資料庫，並邀請更多醫事人員協作以提升完整性。此外，應用範疇不侷限於體檢報告，也可納入血液培養、生化抽血及病理檢查等需長期追蹤之項目，以強化臨床追蹤的深度。
